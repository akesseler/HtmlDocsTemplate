Overview

Write an introduction and put it right here. As next follow the steps below to add and configure your additional pages. Finally, write content of each page and link them to the navigation bar.

Add New Page

As first, create a new HTML page and add it to the project. Thereafter, insert a link to the new page into the navigation bar. See code snippet below how to do it.

Add Page Link

Below an example of how to add a link to the navigation bar. The navigation bar can be found in page index.html. 

<a id="menu-item-id" href="#" onclick="showContent(this, 'menu-item-page-name.html');">Menu Text</a>

Chapter Referencing

Referencing other chapters, like Licensing, can be accomplished by putting a link into the text as shown below. 

<a id="licensing" href="#" onclick="showContent('licensing', 'licensing.html');">Licensing</a>

Section Referencing

It is possible to link sub-sections into the navigation bar. For this purpose just add a page and configure it as shown below.

Provide Headings

Write your page content, which might look like shown below. 

<h2>Sub-Page Title</h2>
<p>The introduction text of this page.</p>

<h3 id="sub-section-id">Sub-Section Title</h3>
<p>Note that the 'sub-section-id' is important for linking this section into the navigation bar.</p>

Repeat adding of sub-section as long as required.

Navigation Linkage

Open page index.html and move to the navigation bar. 

<a id="menu-item-id" href="#" onclick="showContent(this, 'menu-item-page-name.html');">Menu Text</a>
<a id="sub-menu-item-id" class="navsub" href="#" onclick="showContent(this, 'menu-item-page-name.html', 'sub-section-id');">Sub-Menu Text</a>
<a id="sub-menu-item-id" class="navsub" href="#" onclick="showContent(this, 'menu-item-page-name.html', 'sub-section-id');">Sub-Menu Text</a>
<a id="sub-menu-item-id" class="navsub" href="#" onclick="showContent(this, 'menu-item-page-name.html', 'sub-section-id');">Sub-Menu Text</a>

Save all and test it in the browser. Done.
